# Blooket-Cheats-GUI 
Blooket is one of the most popular school games, so I made a Blooket Hacker that gives you everything. In this, you can find Global Cheats that work in all game modes and there are individual ones that work only in a certain game mode. Right now the only way this is accessible is through the extensions Violentmonkey and Tampermonkey.
